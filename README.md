# YAI-17_Toy-Project_Stock-Trading
연세대학교 인공지능학회 YAI 17기 활동을 하며, 겨울 방학 기간 동안 기초 CV 3팀으로 공부를 진행.
이 과정에서 카이스트 딥러닝 강좌인 '딥러닝 홀로서기' 수업을 수강하고, 최종 발표 프로젝트를 진행했습니다.
해당 프로젝트의 주제로 인공지능을 활용한 주가 예측 모델을 만들 때, 주가 데이터가 Sequence Data임으로 LSTM을 활용해 학습을 진행하는데, 겨울 방학 동안 공부한 CV 모델들을 주가 모델에 적용했을 때 어떠한 결과를 도출할지 궁금해졌습니다.
이에 주가 예측 모델로 LSTM과 ResNet을 활용했을 때, 각각 성능이 어떤지 비교해 보는 것으로 주제를 설정했습니다.

본 Github Repository는 위 주제에 대해 연구 / 실험을 진행하면서 제작한 Dataset, 실험 코드, 발표 자료 등 모든 자료들을 포함하고 있습니다.
